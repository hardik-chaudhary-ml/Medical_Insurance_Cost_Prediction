🔥 Insurance Cost Predictor (End-to-End ML Project)





This project is a Machine Learning-based web application that predicts medical insurance costs based on user inputs such as age, BMI, smoking habits, and region.




It demonstrates a complete ML pipeline:



«Data → Preprocessing → Model Training → Evaluation 



\---



🎯 Key Highlights



\- ✅ End-to-End Machine Learning Project

\- ✅ Interactive Web App using Streamlit

\- ✅ Real-time Predictions

\- ✅ Data Visualization (Cost vs Age graph)

\- ✅ Clean UI with user insights

\- ✅ Production-ready project structure



\---



🧠 Machine Learning Details



\- Model Used: Random Forest Regressor

\- Performance: \~74% R² Score

\- Reason for Selection:

&#x20; - Handles non-linear relationships effectively

&#x20; - Reduces overfitting via ensemble learning

&#x20; - Strong generalization on real-world data

📊 Features Explained
